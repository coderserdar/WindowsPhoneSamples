This sample demonstrates how to use Windows.Security.Authentication.OnlineId APIs to authenticate a user with Microsoft account in delegation mode and send acquired tickets to Live Connect APIs using the REST protocol.
Before your application makes requests of the Live Connect APIs to work with Live Connect information, in most cases you must get consent from the user to access that information or to create new objects on behalf of the user.

If you want to register your application, please see the documentation on Developer Center for more details.

To sign out from the application, go to the Settings charm.
