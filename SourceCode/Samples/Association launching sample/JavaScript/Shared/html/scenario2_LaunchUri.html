Clicking on this button will launch the URI entered below with the default handler installed on the system.

Clicking on this button marks the URI as untrusted, which results in a warning prompt being shown prior to launching the URI.

Clicking on this button will launch the 'Open With' dialog that lets the user chose the handler for the URI.

Clicking on this button sets Windows.System.LauncherOptions.DesiredRemainingView to the value selected below. Each of these values expresses a preference for how this app should be sized on screen after the URI launch. It may not be possible to share the screen, in which case the launched app will replace this one.

To see the effect of the various launcher options when launching different types of apps try launching different URI schemes.

All variations use Windows.System.Launcher.launchUriAsync.
