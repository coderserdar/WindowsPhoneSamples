Clicking on this button will launch a .png file with the default handler installed on the system.

Clicking on this button marks the file as untrusted, which results in a warning prompt being shown prior to launching the file.

Clicking on this button will launch the 'Open With' dialog that lets the user chose the handler for the .png file.

Clicking on this button will launch the file picker that lets the user pick the file to launch. The picked file is then launched.

Clicking on this button shows the file picker and sets Windows.System.LauncherOptions.DesiredRemainingView to the value selected below. Each of these values expresses a preference for how this app should be sized on screen after the file launch. It may not be possible to share the screen, in which case the launched app will replace this one.

All variations use Windows.System.Launcher.launchFileAsync.
