Create a file with a .alsdkjs extension. In the OneDrive app, launch that file. This sample sets the DesiredView element within the FileType extension category to "useLess". When you launch the sample from another app, such as the OneDrive app, this preference tells the system to attempt to use a smaller portion of the screen for the sample and leave the rest for other apps.

Create a file with a .alsdkjs extension on PC and upload it to your OneDrive folder. On phone, in the OneDrive app, launch that file. It will trigger this sample app to launch into this page.
